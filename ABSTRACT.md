# Abstract

Concrete Crack Images for Classification is a curated packaging of a concrete crack dataset released on Mendeley Data for reproducible benchmarking.

The source data contains 40,000 RGB image patches at 227x227 resolution split into cracked and non-cracked classes, generated from higher-resolution concrete surface images with varying finish and illumination conditions.

This package preserves publication metadata and adds generated statistics and visualization artifacts for the Large Road Damage DataLake.
