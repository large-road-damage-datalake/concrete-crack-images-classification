# Concrete Crack Images for Classification

## Overview

Concrete Crack Images for Classification is a task-specific package derived from the Mendeley Data dataset.

- Task: classification
- Images: 40000
- Annotations: 40000
- Classes: 2 (cracked, non-cracked)
- Annotation format: image-level labels in class folders

This package contains 227x227 RGB concrete surface patches from METU campus buildings.
